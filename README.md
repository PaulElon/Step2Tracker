# Step 2 Command Center

Local-first Step 2 study operating system built with React, TypeScript, Vite, Tailwind CSS, Plotly, and browser persistence.

## Features

- Premium dashboard for daily task load, upcoming work, category mix, and trend lines
- Daily study planner with CRUD, search, importance ordering, checkbox completion, and legacy Excel migration
- Practice test logging with reflections, action plans, and score trends
- Analytics driven by persisted in-app data
- Optional Excel import for legacy workbook migration

## Development

```bash
npm install
npm run dev
```

## Desktop

```bash
npm run tauri:dev
```

Build the macOS desktop app:

```bash
npm run tauri:build
```

## Updater

Release builds now check for signed updates on launch. Set `STEP2_UPDATER_ENDPOINT` or `STEP2_UPDATER_ENDPOINTS` to a release manifest URL, then build with `TAURI_SIGNING_PRIVATE_KEY` pointing at the local private key generated by `npm run tauri signer generate -- -w ~/.tauri/step2-command-center.key`.

## Verification

```bash
npm run typecheck
npm run lint
npm run build
```
